Current Development

You can also get the latest code from our GitHub Repository. Development code is not guaranteed to be 100% stable, but it may contain features not yet available in official releases.
git clone https://github.com/vufind-org/vufind.git
To learn more about Git, see our Git documentation. This is recommended reading if you plan on making significant customizations to the software!

Legacy Development

The VuFindÂ® 1.x code is no longer under active development, but it is still available in the SourceForge Subversion Repository. It is not recommended that you use this code, but it remains available for historical purposes.
svn export svn://svn.code.sf.net/p/vufind/svn/trunk/ /usr/local/vufind
